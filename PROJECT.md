# doyoudo 网站克隆项目 ✅

## 📋 项目概述

**精确克隆 www.doyoudo.com** — 创意设计软件在线学习平台。基于真实网站 CSS 逆向分析，精确复刻品牌色、布局尺寸、字体系统、组件样式和交互逻辑。

| 特性 | 来源 |
|------|------|
| 品牌色 | `#a2c217` (绿色) — 从原站 CSS 提取 |
| 字体 | `PingFangSC, Microsoft Yahei` — 匹配原站 |
| 图标库 | Remix Icon CDN — 与原站一致 |
| 容器宽度 | `1240px` — 匹配原站 |
| 卡片尺寸 | `294px` 宽 × `166px` 封面高 — 匹配原站 |
| Logo | `assets/images/web-logo.png` — 原站下载 (712×158) |
| Banner | `assets/images/banner.jpg` — 原站下载 (1622×1210) |
| 课程封面 | 原站 Telegram CDN 真实图片 |

## 📄 四个页面

### 1. 首页 (`index.html`)

- **导航栏**：72px 高度，Logo (120px宽) + beta 标签 + 6个菜单项
- **Banner区**：大图 (792×297) + 小图 (396×297)，绿色渐变占位
- **搜索区**：搜索框 + "全部课程"（绿色）+ "付费课程"（橙色）+ "企业合作"（蓝色）
- **筛选栏**：软件 (PS/AE/PR/C4D/AI/AU/FL/LR/FCP) + 难度 (初级/中级/高级)
- **课程卡片网格**：4列 × 294px，真实封面图 + 软件标签 + 难度标签
- **Footer**：120px 高度，Logo + 链接 + ICP备案

### 2. 教程列表页 (`tutorials.html`)

- 与首页相同的搜索 + 筛选区域
- 增加了类型筛选（免费/付费）+ 排序下拉框
- 分页器（省略号逻辑）
- 空状态提示

### 3. 教程详情页 (`tutorial-detail.html`)

- **面包屑**：首页 > 教程 > 课程名称
- **视频播放器**：16:9 黑色播放器模拟，进度条 + 控制条
- **课程信息**：标题/软件/难度/作者/时长/课时/评分
- **Tab 切换**：课程简介 / 课程目录（章节列表，绿色当前高亮）/ 学员评论
- **侧边栏**：课程信息摘要 + 作者卡片
- **相关推荐**：底部课程卡片

### 4. 学习路径页 (`learning-path.html`)

- 页面标题 + 描述
- 软件筛选标签（全部/PS/AE/PR/C4D/AI/AU/FL/FCP）
- 路径卡片：2列，软件图标 + 阶段流程（入门→核心→进阶→精通）+ 课时统计
- 空状态：建设中提示

## 📦 技术说明

- **纯前端静态页面**，无需构建工具，双击即可运行
- 图片资源：Logo、Banner 从原站直接下载；课程封面从 Telegram CDN 下载
- 无封面课程使用软件色渐变占位图
- 所有交互（筛选/分页/Tab/导航）均为前端 JS 实现
- 外部依赖仅 Remix Icon CDN（与原站一致）
- 无需 Google Fonts（原站使用系统字体 PingFang SC）
