# Web 课程项目汇报稿

## 一、项目概述

本项目是一个**创意设计软件在线学习平台**的前端静态页面克隆，目标网站为 www.doyoudo.com。项目完整实现了首页、教程列表页、课程详情页、学习路径页四个核心页面，精确复刻了原站的品牌色、布局尺寸、字体系统、组件样式和交互逻辑。

---

## 二、开发流程与方法论

### 2.1 三阶段开发法

本项目采用"**分析 → 编码 → 验证**"三阶段开发流程：

#### 第一阶段：深度分析与规划
在编写任何代码之前，先完成以下工作：
- **UI 结构拆解**：将页面拆分为导航栏、Banner 区、筛选栏、课程卡片网格、Footer 等独立模块
- **设计系统提取**：从原站 CSS 逆向分析出品牌色 `#a2c217`、容器宽度 `1240px`、卡片尺寸 `294px×166px` 等精确参数
- **文件结构规划**：设计模块化的 CSS 和 JS 目录结构

#### 第二阶段：分层编码实现
按照"**设计令牌 → 基础组件 → 页面组装 → 交互细节**"的顺序逐层实现：
1. 先完成全局重置样式和颜色/阴影/圆角规范
2. 再开发导航栏、卡片等通用组件
3. 然后组装各个页面
4. 最后添加筛选、搜索、Tab 切换等交互

#### 第三阶段：测试与验证
对照原站逐一验证：
- 布局还原度（尺寸、间距、对齐）
- 颜色准确性（品牌绿、文字色、背景色）
- 响应式适配（桌面/平板/手机三端）
- 交互完整性（筛选/搜索/分页/URL 同步）

## 三、技术栈选型

### 3.1 核心技术栈

| 技术类别 | 选型 | 说明 |
|---------|------|------|
| **HTML** | HTML5 语义化标签 | 使用 `<header>`, `<nav>`, `<main>`, `<section>`, `<footer>` 等语义标签 |
| **CSS** | 原生 CSS3 | 模块化拆分、Flexbox 布局、媒体查询响应式 |
| **JavaScript** | ES6+ 原生 JS | 箭头函数、模板字符串、解构赋值、模块化架构 |
| **图标库** | Remix Icon CDN | 与原站一致的图标解决方案 |
| **字体** | 系统字体栈 | `PingFang SC, Microsoft YaHei, sans-serif` |

### 3.2 技术选型理由

- **零构建工具依赖**：无需 npm、webpack、Vite，双击 `index.html` 即可运行，便于演示和部署
- **纯前端静态实现**：适合课程作业的展示场景，无需配置服务器环境
- **模块化架构**：虽然不用构建工具，但仍通过文件拆分实现代码组织

## 四、架构设计思想

### 4.1 CSS 模块化架构

将样式按**功能模块**而非页面进行拆分，共 8 个 CSS 文件：

```
css/
├── reset.css          # 全局重置和基础样式
├── navbar.css         # 导航栏样式
├── home.css           # 首页/Banner/搜索/筛选区样式
├── cards.css          # 课程卡片通用样式（多页面复用）
├── detail.css         # 课程详情页专用样式
├── paths.css          # 学习路径卡片样式
├── footer.css         # 页脚样式
└── responsive.css     # 响应式适配样式
```

**设计思想**：每个页面只引用必需的 CSS 模块，减少冗余加载。

### 4.2 JS 分层架构

采用"**数据层 → 工具层 → 组件层 → UI 层 → 业务逻辑层**"五层架构：

```
js/
├── data.js            # 数据层：课程数据、路径数据、软件列表
├── utils.js           # 工具层：数据访问、URL 生成、DOM 操作封装
├── components.js      # 组件层：课程卡片 HTML 生成器
├── ui.js              # UI 层：导航/搜索/Footer 初始化
├── home.js            # 业务层：首页热门课程渲染
├── tutorials.js       # 业务层：列表页筛选/搜索/分页
├── detail.js          # 业务层：详情页 Tab 切换/章节渲染
├── paths.js           # 业务层：路径页卡片渲染
├── tab-switch.js      # 特效：Tab 栏切换
└── carousel.js        # 特效：轮播图
```

**设计思想**：
- **单一职责原则**：每个文件只负责一个功能领域
- **数据驱动**：数据与视图分离，修改数据即可更新界面
- **可复用性**：通用工具函数和组件可在多页面复用

### 4.3 数据驱动模式

```javascript
// 1. 数据层定义课程数据
const coursesData = [
  {
    id: 1,
    title: '最容易听懂的 PS 入门教程',
    software: 'ps',
    difficulty: 'beginner',
    // ...
  }
];

// 2. 工具层提供数据访问接口
function getCourses(filters) {
  // 根据筛选条件过滤数据
}

// 3. 组件层将数据转换为 HTML
function generateCourseCard(course) {
  return `<div class="course-card">...</div>`;
}

// 4. 业务层调用并渲染
renderCards(getCourses({software: 'ps'}));
```

## 五、核心功能实现

### 5.1 智能筛选系统

支持多维度组合筛选：
- **软件类型**：PS/AE/PR/C4D/AI/AU/FL/LR/FCP
- **难度等级**：初级/中级/高级
- **课程类型**：全部/免费/付费
- **排序方式**：最新/最热/评分

筛选状态实时同步到 URL 参数，支持分享和刷新保留状态。

### 5.2 搜索功能

- 支持标题、作者、软件名称模糊匹配
- 搜索结果高亮显示
- 回车键快速触发搜索
- 搜索关键词同步到 URL

### 5.3 分页器

- 智能省略号逻辑（当前页附近显示，远处省略）
- 上一页/下一页按钮
- 页码跳转
- 空状态提示

### 5.4 Tab 切换系统

课程详情页的三 Tab 切换：
- 课程简介
- 课程目录（章节列表）
- 学员评论

支持点击切换和自动轮播（5 秒间隔），鼠标悬停暂停。

### 5.5 响应式设计

三个断点完整覆盖：

| 断点 | 设备 | 适配策略 |
|------|------|----------|
| >1024px | 桌面端 | 4 列卡片，1240px 容器，72px 导航 |
| 471-1024px | 平板端 | 3 列卡片，导航换行，Banner 缩小 |
| ≤470px | 手机端 | 1 列卡片，搜索按钮全宽，Footer 堆叠 |
