# doyoudo 网站克隆任务

## 任务概述

你要克隆的网站是 **www.doyoudo.com**，一个创意设计软件在线学习平台。

目标：生成与目标网站完全一致的前端静态页面，包含以下四个关键页面：
1. **首页** — 导航栏、轮播区/Banner、课程卡片列表
2. **教程列表页** — 按软件/难度/作者筛选的课程展示
3. **教程详情页** — 视频播放器、课程信息、相关推荐
4. **学习路径页** — 系统性学习路线图

## 技术约束

- 克隆 **仅限前端部分**，功能性的播放接口、用户登录、评论等后端逻辑 **不需要**
- 除图片/视频资源外，**所有代码必须自包含**，引用原站的外部资源需显式注释说明
- 目标网站为 SPA（前端 React 框架渲染），直接爬取 HTML 不可行，需：
  1. 手动分析 UI 结构和样式模式
  2. 用 Tailwind CSS + HTML/CSS/JS 重构静态页面
- 响应式适配：移动端（375px）、平板（768px）、桌面（1280px+）需**完整覆盖**
- 采用语义化 HTML 标签，确保无障碍可访问性

## 执行步骤

### 阶段一：分析与计划（深度拆解）

在**一行代码都不写之前**，必须先完成分析，并按以下清单输出结果：

1. **UI 组成结构分析**：
   - Header（Logo + 导航菜单 + 搜索）
   - Hero 区域（标语 + CTA）
   - 筛选栏（软件分类 + 难度 + 作者）
   - 课程卡片网格（缩略图 + 标题 + 作者 + 难度标签 + 观看数）
   - Footer（链接 + 备案信息）
   - 找原页面截图对照，逐一拆解模块

2. **设计系统提取**：
   - 字体系统（标题 h1/h2/h3、正文、标签等字号与字重）
   - 颜色系统（主色调、背景色、文字颜色、边框色、悬浮态）
   - 间距系统（margin/padding 的常用值）
   - 圆角与阴影规范（卡片圆角、悬浮阴影）
   - 动画/过渡效果规律

3. **生成克隆计划**：
   - 将每个页面拆解为可独立实现的组件列表
   - 建立依赖顺序与优先级
   - 识别重复使用的全局样式设计模式
